# OYO Hotel Data Analysis Dashboard

## 📊 Project Overview
This project analyzes OYO hotel data to uncover insights related to pricing, ratings, locations, and customer preferences.

## 🛠 Tools Used
- Power BI
- Advanced Excel
- Python (Google Colab)

## 📁 Files Included
- Dataset (Excel)
- Power BI Dashboard (.pbix)
- HTML Report
- Project Documentation

## 📈 Key Insights
- Price vs Rating relationship
- Location-based performance
- Customer trends and patterns

## 📌 Features
- Interactive dashboard
- Filters and slicers
- Data-driven insights

## 🚀 How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore the dashboard
